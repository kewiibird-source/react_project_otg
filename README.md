## 📌 프로젝트 소개

> 혼자서도 술을 즐겨 마시는 사람들의 모임 SNS

혼술을 하면서 자기만의 꿀정보나 레시피를 공유하고 싶을때,
가끔은 내가 갔던 술집이 너무 마음에 들어서 공유하고 싶을때,
내가 만들어보고 싶은 레시피나 정보를 한 플렛폼에서 볼 수 있으면 얼마나 좋을까? 

인스타 릴스, 유튜브 쇼츠로 많은 꿀팁·레시피가 많이 올라옵니다.
하지만 매번 저장해놓고 찾아보기 귀찮은게 단점이였죠
아예 술 전용 SNS가 있으면 좋을 것 같다는 생각을 했습니다. 

원하는 게시물을 저장해놓고, 해시태그를 이용해 검색할 수 있는 시스템

내가 좋아하는 술, 장소 등을 마음껏 공유할 수 있고
혼자서 마시는 술이지만 잔 너머에는 여러 사람들과 함께한다는 뜻으로 만들었습니다.
